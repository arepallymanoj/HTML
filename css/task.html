CSS SELECTORS

In CSS, selectors are patterns used to select and style specific HTML elements.

TYPES OF CSS SELECTORS

  1. Simple
  2. Combinator
  3. Attribute
  4. Pseudo Classes
  5. pseudo Elements

SIMPLE SELECTORS

Simple Selectors in Different Types

Element Selector:

Select element according to their tag

Example:(p)

Class Selector:

Select all elements with a specific class attribute

Example:(.)

Id Selector:

Select Single element according with a specific Id attribute

Example:(#)

Group Selector:

Select Multiple element by seperating each sector with a comma

(p,ab,#xy)

Universal Selector:

Select all element in a document.

Example:(*)
